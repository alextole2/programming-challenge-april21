# Movies Lens Client (movies-frontend)

## Technologies:

* Vue JS
* JavaScript
* HTML
* CSS/SASS
* Quasar Framework

## Install:

* npm install
* npm i yarn
* npm install -g quasar-cli

## Start the app in development mode (hot-code reloading, error reporting, etc.):

* quasar dev

## Build the app for production:

* quasar build

## Url:

* http://localhost:8080
